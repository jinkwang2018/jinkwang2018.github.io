# hosts file
> /etc/hosts 파일이다.

> 도메인의 ip를 개인적으로 변경해야 할 때

# Domain
> ip를 연결해준다.


> 하나의 도메인은 하나의 server를 가질 수 있다.

# serve domain
> 도메인 앞에 다른 이름을 두어 여러 ip를 하나의 도메인에 연결 할 수 있다.
> ex) news.corin2.site
